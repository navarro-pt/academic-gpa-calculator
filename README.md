# Student Grade & GPA Calculator (C++) 🎓

A robust terminal-based tool developed to calculate modular averages and overall course GPA. 

## ✨ Key Features
- **Dynamic Calculation:** Handles multiple subjects and varying numbers of modules per subject.
- **Robust Validation:** Prevents invalid grades (outside 0-20) or module counts.
- **Scalable Design:** Built using constants for easy configuration of the number of subjects and modules.
- **Clean UI:** Uses a menu-driven interface with console clearing for a better user experience.

## 💻 Technical Highlights
- **Data Structures:** Implementation of `struct` for organized data management.
- **Input Handling:** Use of `while` loops for continuous input validation.
- **Global Constants:** Implementation of best practices for code maintainability.

## 🛠️ Compatibility
- **OS:** Windows Only (Uses `windows.h` and `system` commands).
- **Compiler:** Any C++ compiler (MinGW, Visual Studio, Dev-C++).
